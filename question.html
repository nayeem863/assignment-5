Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML = retrieve html content
innerText = retrieve visible text inside an element
textContent = retrieve entire text content including hidden text

Question 3:What is event delegation in the DOM?

Question-4: What is event bubbling in the DOM?

Event starts from the child element and propagate to its parent element
